# Vehicle Document Verification System Using QR Code

## Third Year Group Project

This project is a web-based system developed to verify vehicle documents using QR Code technology.

### Features
- Vehicle owner registration
- Upload vehicle documents
- Generate QR Code
- Officer QR code scanning
- Document verification system

### Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

### Developed For
Third Year Undergraduate Group Project
